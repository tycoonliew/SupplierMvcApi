# SupplierMvcApi

## Summary
Manages products and suppliers for companies. Built on ASP.Net MVC.

## Routes
**Products**

```GET /api/products```
returns list of Products

```POST /api/products```
takes a body of type Product
adds it to the database

**Suppliers**

```GET /api/suppliers```
returns list of Suppliers

```POST /api/suppliers```
takes body of type Supplier
adds it to database
